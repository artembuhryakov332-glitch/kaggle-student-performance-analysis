# Анализ успеваемости студентов (Kaggle Student Performance)

Краткое описание: исследование факторов, влияющих на оценки студентов, с использованием методов кластеризации и понижения размерности.

## Источник данных
* Данные взяты с Kaggle: https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

## Используемые технологии
* Python (Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib)
* Метод главных компонент (PCA) для снижения размерности
* Кластеризация (K-Means)
* Алгоритмы LinearRegression, KNN, DesionTree, RandomForest

## Признаки и их расшифровки:
## Описание признаков датасета (Dataset Features)

Проект исследует влияние различных факторов (демографических, социальных и подготовительных) на успеваемость студентов по трем ключевым предметам. Оригинальный датасет содержит **8 признаков (столбцов)**:

### Категориальные признаки (Факторы влияния):

1. gender (Пол) — номинальный признак:
   * female — женский
   * male — мужской
2. race/ethnicity (Раса / Этническая группа) — замаскированный категориальный признак. Разделен на 5 гипотетических групп:
   * group A, group B, group C, group D, group E
3. parental level of education (Уровень образования родителей) — порядковый признак, показывающий наивысшую степень образования:
   * some high school — неоконченная средняя школа
   * high school — оконченная средняя школа
   * some college — неоконченное высшее образование / колледж
   * associate's degree — степень ассоцииата (среднее профессиональное образование)
   * bachelor's degree — степень бакалавра
   * master's degree — степень магистра
4. lunch — тип обеда:
   * standard — стандартный обед (полная стоимость)
   * free/reduced — бесплатный обед или обед по сниженной цене (индикатор более низкого дохода семьи)
5. test preparation course (Курс подготовки к экзаменам) — показатель самостоятельной подготовки студента перед тестированием:
   * none — курс подготовки не проходился
   * completed — курс подготовки успешно завершен

### Количественные признаки (Результаты экзаменов / Целевые переменные):

Все оценки представлены в виде баллов по 100-балльной шкале (непрерывные числовые величины):

6. math score — балл за экзамен по математике.
7. reading score — балл за экзамен по чтению.
8. writing score — балл за экзамен по письму.

## Основные результаты
* Выделено 3 устойчивых кластера: «Успешные», «Средние» и «Плохие» студенты (сегментация данных)
* Также было проведено разбиение на 2 кластера («Успешные» и «Плохие» студенты), так как такое разбиение более "естественное" для этого набора данных
* Самым успешным алгоритмом ML оказался LinearRegression(0.23).Признаки имеют линейные зависимости(целевые переменные math score, writing score, reading score; lunch, test preparation course) и закономерности (порядок в переменных race/ethnicity,parental level of education).Такой показатель считается нормой для этих данных, поскольку сами признаки не являются информативными (шумные по своей природе)
